16.07

1 安装上传组件
yum install -y lrzsz

2
cd /usr 
输入rz命令，  分别选java.gz 和 tomcat.gz上传
rz 上传

3 创建目录
mkdir /usr/java
mkdir /usr/tomcat
mkdir /usr/redis
解压tomcat 9
tar -zxvf jdk-11.0.13_linux-x64_bin.tar.gz -C /usr/java
tar -zxvf jre.tar.gz -C /usr/java
tar -zxvf /usr/apache-tomcat-9.0.60.tar.gz -C /usr/tomcat
tar -zxvf /usr/redis-6.2.6.tar.gz -C /usr/redis

4 安装c语言依赖
yum install gcc-c++
编译redis源码
cd /usr/redis/redis-6.2.6 
make MALLOC=libc
安装redis
make install PREFIX=/usr/redis/redis-6

5 启动，验证是否安装成功
/usr/redis/redis-6/bin/redis-server
正常关闭的代码
/usr/redis/redis-6/bin/redis-cli shutdown

6 把conf配置 文件复制过来
cp /usr/redis/redis-6.2.6/redis.conf /usr/redis/redis-6/bin
把redis.config 里的属性
cd /usr/redis/redis-6/bin
yum install vim -y
vim redis.conf
/daemon可以查找 deamon的位置，按 i 进入编辑 改为 daemonize yes

看进程中，筛选出redis相关的
ps -aux | grep redis
