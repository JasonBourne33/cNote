16.07

1 安装上传组件
yum install -y lrzsz

2
cd /usr 
输入rz命令，  分别选java.gz 和 tomcat.gz上传
rz 上传

3 创建目录
mkdir /usr/java
mkdir /usr/tomcat
mkdir /usr/redis
解压tomcat 9
tar -zxvf jdk-11.0.13_linux-x64_bin.tar.gz -C /usr/java
tar -zxvf jre.tar.gz -C /usr/java
tar -zxvf /usr/apache-tomcat-9.0.60.tar.gz -C /usr/tomcat
tar -zxvf /usr/redis-6.2.6.tar.gz -C /usr/redis

4 安装c语言依赖
yum install gcc-c++
编译redis源码
cd /usr/redis/redis-6.2.6 
make MALLOC=libc
安装redis
make install PREFIX=/usr/redis/redis-6

5 启动，验证是否安装成功
/usr/redis/redis-6/bin/redis-server
正常关闭的代码	
/usr/redis/redis-6/bin/redis-cli shutdown

6 后端启动
把conf配置 文件复制过来
cp /usr/redis/redis-6.2.6/redis.conf /usr/redis/redis-6/bin
把redis.config 里的属性
cd /usr/redis/redis-6/bin
yum install vim -y
vim redis.conf
/daemon可以查找 deamon的位置，按 i 进入编辑 改为 daemonize yes
用命令后端启动
/usr/redis/redis-6/bin/redis-server redis.conf

看进程中，筛选出redis相关的
ps -aux | grep redis
强行关闭进程
kill -9 13211
正常关闭 
/usr/redis/redis-6/bin/redis-cli shutdown

进入客户端
/usr/redis/redis-6/bin/redis-cli

7 配置密码
vim /usr/redis/redis-6/bin/redis.conf
找到requirepass
/requirepass ,按n找下一个
找到# requirepass foobared，删掉 # 解开注释
可以把密码改成123 requirepass 123，保存退出 :wq
带密码登录 -a 表示授权 123是密码
/usr/redis/redis-6/bin/redis-cli -a 123
rm /usr/redis/redis-6/bin/.redis.conf.swp


8 redis destop manage登录
先在服务器开防火墙和端口
systemctl start firewalld
firewall-cmd --zone=public --add-port=6379/tcp --permanent
systemctl restart firewalld.service
firewall-cmd --reload 	//添加规则后重载防火墙
netstat -tnlp 查看端口
修改redis.conf配置
vim /usr/redis/redis-6/bin/redis.conf
找到bind 127.0.0.1， 加注释 #bind 127.0.0.1
重启redis，先shutdown，再启动
/usr/redis/redis-6/bin/redis-cli shutdown
/usr/redis/redis-6/bin/redis-server redis.conf

打开desktop manage
名字aliyun
地址 47.112.147.20：6379
